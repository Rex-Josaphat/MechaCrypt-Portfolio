# MechaCrypt: Hardware-Encrypted Messaging System

Welcome to **MechaCrypt**. This is an experimental hardware communication system that transmits encrypted data using physical actuation instead of traditional signaling. This repository contains all files and configurations required to build and deploy the MechaCrypt project website.
